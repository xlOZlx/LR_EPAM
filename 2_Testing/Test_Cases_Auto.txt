Сайт по аренде автомобилей – www.rentalcars.com


1. Бронирование авто без указания страны/города/места
	Шаги: заходим на главную страницу www.rentalcars.com, пропускаем ввод 
	страны/города/места, для приличия указываем дату бронирования, нажимаем поиск.
	Ожидаемый результат: пользователь получает уведомление о необходимости указать 
	данные параметры.

2. Указать дату получения авто позже даты возврата
	Шаги: заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения: 27 сен 2019, а дату возврата на месяц раньше: 
	27 авг 2019.
	Ожидаемый результат: сайт не дает пользователю такой возможности ошибиться.

3. Указать дату получения равную дате воврата с точностью по часам и минутам
	Шаги: заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата 27 сен 2019 10:00.
	Ожидаемый результат: пользователь ролучает уведомление о том, что между 
	датой получения и возврата должна быть разница минимум в один час.

4. Установить возраст водителя отрицательным или равным 0
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, указываем возраст водителя -1.
	Ожидаемый результат: пользователь получает уведомление о том, что возраст 
	водителя должен быть числом.

4. Посмотреть список дуступных мест получения авто
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, ставим галочку по возрасту, 
	если соответсвует стандарту или убираем и вводим необходивый возраст, выбираем 
	из выпадающего списка все места, нажимаем кнопку "Найти".
	Ожидаемыый результат: Потзователь попадаеь на станису со списком возможных мест 
	получения авто и указанием кол-ва авто на этой точке.

5. Найти авто в аренду
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, ставим галочку по возрасту, 
	если соответсвует стандарту или убираем и вводим необходивый возраст, нажимаем 
	кнопку "Найти".
	Ожидаемый результат: пользователь получает список доступных для аренды авто.

6. Орендовать авто без указания данных пользователя
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, ставим галочку по возрасту, 
	если соответсвует стандарту или убираем и вводим необходивый возраст, нажимаем 
	кнопку "Найти". Из списка выбрать понравившийся авто и нажать кнопку "Бронировать".
	После попадания на страницу, пользователь, по желанию, может выбрать доп. услуги, 
	выбрать иметь защиту на авто (доп. оплата) или нет. После пользоваель попадает на 
	страницу, где вводятся его данные.
	Ожидаемый результат: Пользователю не дают орендовать без указания данных, 
	и просят его это сделать.

7. Орендовать авто с дополнительными услугами.
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, ставим галочку по возрасту, 
	если соответсвует стандарту или убираем и вводим необходивый возраст, нажимаем 
	кнопку "Найти". Из списка выбрать понравившийся авто и нажать кнопку "Бронировать".
	После попадания на страницу, пользователь, путем выставления галочек в отведенном 
	месте выбирает необходимые ему доп. услуги.
	Ожидаемый результат: стоимость выбранных дополнительных услуг входит в состав 
	итоговой цены.

8. Просмотр возможных авто от определенного поставщика
	Шаги:заходим на страницу www.rentalcars.com, вводим необходимые данные по месту 
	получения, вводим дату получения и дату возврата, ставим галочку по возрасту, 
	если соответсвует стандарту или убираем и вводим необходивый возраст, нажимаем 
	кнопку "Найти". Далее с лева располагауюся доп. фидьтры, шде мы указываем 
	интерисующих пользователя поставщиков из предложенных.
	Ожидаемый результат: через небольшой промежуток времени список обновляется и в нем 
	представлены авто от выбранных поставщиков.

10. Возможность связаться с представителями
	Шаги:заходим на страницу www.rentalcars.com, внизу страницы в подразделе 
	"Помощь и информация" пользователь выбирает "Связаться с нами".
	Ожидаемый результат: пользователь попадает на страницу с часто задваемыми 
	вопромами и разделами по интересующим его вопросом и процессам, с 
	возможностью связаться с консультантом.
